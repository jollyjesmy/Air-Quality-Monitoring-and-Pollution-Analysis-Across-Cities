# Air-Quality-Monitoring-and-Pollution-Analysis-Across-Cities
ABOUT PROJECT: Air Quality Monitoring and Pollution Analysis Across Cities — A data analytics project that collects, cleans, and analyzes air quality index (AQI) and pollutant concentrations across multiple cities. Includes Power BI dashboards, statistical insights, and data-driven visualizations to understand pollution patterns and highlight high-risk zones.
KEY FEATURES: City-wise AQI comparison, Pollutant trend analysis (PM2.5, PM10, NO₂, SO₂, CO, O₃), Interactive Power BI dashboard, Cleaned datasets and DAX measures included, City-wise AQI comparison
TOOLS USED: Power BI, Excel / Power Query, DAX
